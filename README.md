# Ping Pong Game 🏓

A classic Ping Pong (Pong) game built using **Python and Pygame**, featuring player controls, AI opponent, and real-time game physics.

---

## GOAL

Play the game on loop. In the future other features will be added to the game like sound effects, levels, music, score etc. 

If you played this, let me know, and I hope you enjoyed it :)

---

## Overview

This project recreates the classic Pong game, where the player competes against an AI-controlled opponent.  

It demonstrates core game development concepts such as:
- Game loops
- Collision detection
- Basic AI behavior
- Real-time input handling

---

## Features

- Player-controlled paddle using keyboard input (↑ ↓ keys)  
- AI opponent that tracks the ball position  
- Ball physics with collision detection (walls and paddles)  
- Ball reset logic when a point is lost  
- Smooth rendering at 60 FPS  

---

## Tech Stack

- **Python** – Core programming language  
- **Pygame** – Game development library for rendering and input handling  

---

## Key Skills Demonstrated

- Game loop implementation using Pygame  
- Collision detection using rectangles  
- Event-driven programming (keyboard input handling)  
- Basic AI logic (opponent tracking ball position)  
- State updates and real-time rendering  
- Debugging and structuring game logic
